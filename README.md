## 🚀 How to Run

```bash
# Compile using GCC
gcc program.c -o program.exe

# Run the executable
./program.exe
```

### 📌 Example Program (Factorial)

```c
#include <stdio.h>
int main() {
    int n, fact = 1;
    printf("Enter a number: ");
    scanf("%d", &n);
    for(int i=1; i<=n; i++) {
        fact *= i;
    }
    printf("Factorial = %d", fact);
    return 0;
}
```
