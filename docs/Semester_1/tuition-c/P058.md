## Source Code
```c
#include <stdio.h>

void displayASCII()
{
    int i;
    printf("ASCII Code\t\tCharacter");
    for (i = 'A'; i <= 'Z'; i++)
        printf("\n%c\t\t%d", i, i);
}

int main()
{
    displayASCII();
    return 0;
}
```
